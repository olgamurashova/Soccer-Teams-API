## Soccer Teams API

### Project overview

This is a basic API that serves information about two soccer teams. API routes and routers live in routesRouters.js file. Soccer teams information is stored in two files: teamLions.js and teamTigers.js. The server information is stores in server.js file. 

#### [Routes](#routes)

```/teamlions/players```

```/teamtigers/players```

* GET / to get an array of all players.
* GET /:id to get a single player by id.
* POST /:id to create a new player.
* PUT /:id to update a single player by id.
* DELETE /:id to delete a player by id.
